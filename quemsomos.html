Quem Somos

Quitanda Virtual é uma loja (e-commerce) fictícia criada como estudo de caso para o curso "Desenvolvimento de uma Loja Virtual com Bootstrap 5 e ASP.NET 5" disponibilizado gratuitamente pelo Prof. Ricardo Maroquio em seu canal do Youtube.

O código fonte pode ser obtido via GitHub, conforme orientações apresentadas ao longo dos vídeos do curso no Youtube. Este estudo de caso pode ser utilizado livremente como base para outras lojas virtuais, sem ônus para o usuário.
